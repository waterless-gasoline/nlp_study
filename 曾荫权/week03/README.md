# bert.py 与训练模型分类器
## 优点
文本分类的高准确率 精准的上下文的理解能力
## 缺点
部署成本高 可解释性差 内存占比大

# prompt.py LLM
## 优点
能够理解较为复杂的需求 灵活度高可通过较少的样本学习和调整提示词分类处理新需求
## 缺点
推理速度慢 成本高依赖外部服务 输出不稳定易泛化

# regex_rule.py 正则表达式
## 优点
精确匹配关键词 规则清晰 一个类别对应一个规则 易于修改
## 缺点
维护成本过高 依靠关键词匹配无法处理规则之外和过于复杂的需求

# tfidf_ml.py tf-idf
## 优点
通过特征提取和传统机械学习计算效率高适合实时应用 数据集要求较低 实现过程简单 资源消耗较低
## 缺点
无法理解词序和上下文 无法处理未训练过的词义 依赖人工特征标注
